About TreeSeek

Often when I'm walking, I see a tree I like. I can't always identify species by sight, but I'm usually curious what I'm looking at. This web application was built to help with that.

The tree data comes from city open data sites. Right now TreeSeek includes data for:

  • Toronto
  • Ottawa
  • Montreal
  • Calgary
  • Waterloo
  • Boston
  • Markham
  • Oakville
  • Peterborough
because those are some of the cities I have found myself in wandering and wondering about trees.

To use the map: tap a tree marker to view details, tap a row in the nearby trees table to highlight that tree on the map, and use the map controls to recenter or go full screen. Panning the map usually loads nearby trees automatically. If it does not, tap Load more trees.

Controls: ⛶ Full screen, 📍 Recenter, 🌳 Load more trees.

Mapping by Leaflet. Tile hosting by OpenFreeMap.

Source code and issue tracking live on GitHub.
